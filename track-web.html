Направление 01

Web / Mobile

Создай продукт, которым люди будут пользоваться каждый день — в браузере или в кармане.

ReactNode.jsFlutterNext.jsREST API

О направлении

Web/Mobile — самое широкое направление хакатона. Сюда подходят команды, которые хотят сделать что-то конкретное и пощупать руками: сервис, приложение, инструмент. Не важно, работаешь ли ты с фронтендом, бэкендом или занимаешься мобилками — главное, чтобы на демо было что запустить и показать.

Жюри оценивает не стек, а пользу: решает ли продукт реальную проблему, насколько удобен интерфейс, можно ли это масштабировать.

Тематика задач

Конкретная задача выдаётся в первый час хакатона. Прошлые темы помогут понять уровень:

  • Платформа для совместного планирования командных задач без аккаунтов
  • Мобильный помощник для выбора маршрута общественным транспортом в офлайн-режиме
  • Веб-дашборд для агрегации событий из нескольких городских источников
  • Инструмент быстрого онбординга новых сотрудников для малого бизнеса
